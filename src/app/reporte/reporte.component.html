Auto MPG

Este conjunto de datos es una versión ligeramente modificada del conjunto de datos proporcionado en la biblioteca StatLib. De acuerdo con el uso de Ross Quinlan (1993) en la predicción del atributo "mpg", se eliminaron 8 de las instancias originales porque tenían valores desconocidos para el atributo "mpg". El conjunto de datos original está disponible en el archivo "auto-mpg.data-original".
